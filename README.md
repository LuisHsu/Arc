# Arc
A Rambling Compiler

### Prerequisite

g++ >= 4.9

cmake >= 2.8

### Compile

Go to "build" directory 
	
	cd build
	
Configure with CMake
	
	cmake ..
	
Compile with make
	
	make
	
###
    
### Usage

Arc compiler
	
    arc [input files]
	
Luison generator
	
    luison [rule file]

#### Rule example is in misc/GRAMMER.txt
